# ng-event-tracking

Sample web application for tracking Angular-related events

## Quick Start

In the terminal: 

`npm install`

`npm run server`

`npm start`


Then navigate to 

`http://localhost:4200`

and login with

User: `bradgreen`

Pass: `(any)`
